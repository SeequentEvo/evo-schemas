### time-domain-electromagnetic-time-gates (v1.0.0)
Time domain electromagnetic time gate information.

| Property | Type | Description | Flags |
|---|---|---|---|
| id | Number | Gate identifier | ✅ |
| open | Array[Number] | Array of time gate open intervals (s). Time open gates, as consecutive times relative to the Tx Turn off time+start time. | ✅ |
| close | Array[Number] | Array of time gate close intervals (s). Time close gates, as consecutive times relative to the Tx Turn off time+start time. | ✅ |
