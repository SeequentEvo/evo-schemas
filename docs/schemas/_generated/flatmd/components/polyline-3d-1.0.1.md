### polyline-3d (v1.0.1)
This component describes a 3D polyline.

| Property | Type | Description | Flags |
|---|---|---|---|
| begin | Integer | Index of the first line segment in lines-3d. | ✅ |
| count | Integer | The number of line segments. | ✅ |
| closed | String | Closed or open. | ✅ |
