# nan-categorical

<SchemaUri uri="schema/components/nan-categorical/1.0.1/nan-categorical.schema.json" />

The `nan-categorical` component defines values used to designate not-a-number (NaN) for integral data.
These can be used as error codes or to indicate missing data in categorical attributes.

**Used by:** [category-attribute](category-attribute.md), [category-ensemble](category-ensemble.md), [category-time-series](category-time-series.md), [date-time-attribute](date-time-attribute.md), [integer-attribute](integer-attribute.md).
